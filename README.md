## 📋 Sobre o Projeto

Painel interno desenvolvido para um escritório de advocacia, com o objetivo de 
visualizar e gerenciar dados de agendamentos jurídicos a partir de uma base de 
dados em JSON.

A aplicação conta com dashboard de métricas, tabela com busca e filtros em tempo 
real, paginação e exportação de dados em CSV e PDF.

## 🚀 Tecnologias

- **Front-end:** React + Vite, Recharts, jsPDF
- **Back-end:** Node.js, Express
- **Dados:** JSON estático (mock)

## ✨ Funcionalidades

- Cards de KPI com totais e taxa de realização
- Gráfico de barras com distribuição por status
- Gráfico de linha com evolução mensal dos agendamentos
- Tabela com busca textual e filtros
- Paginação dos resultados
- Exportação em CSV e PDF respeitando filtros ativos
